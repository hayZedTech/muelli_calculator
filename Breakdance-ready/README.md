# Breakdance-ready package

This folder contains a copy-ready version of the Muelli Calculator landing page for use in Breakdance or any static hosting setup.

## What is inside
- index.html — the main page
- assets/ — all CSS, JavaScript, images, partials, and fonts used by the page

## How to use it
1. Copy the entire Breakdance-ready folder to your project or hosting directory.
2. Open index.html in a browser to preview it.
3. If you are using Breakdance, upload or include this folder in the same location as your site assets and reference the page from your builder.

## Notes
- The calculator is fully interactive.
- The consultation form opens WhatsApp or email depending on the configuration.
- The sample WhatsApp number can be changed in the form attribute inside index.html.

## Recommended client handoff
This version is intended to be a clean, portable package that can be handed to a client without needing extra setup.
